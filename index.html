Dans ce projet de groupe, nous devions réaliser un exercice en trio pour tester les commandes git push et git pull. L'objectif était de travailler sur trois branches distinctes, chaque branche étant dédiée à une partie spécifique de l'index. À la fin, nous devions fusionner nos modifications dans l'ordre sur la branche main.
les rôles pour le projet:

  • ELIAS : les cards
  • RAYAN : le footer
